# 人脸数据存放说明

## 目录结构

在此目录下为每个需要识别的人创建独立文件夹，文件夹名即为识别显示的姓名。

```
data/known/
├── README.md          # 本说明文件
├── 张三/              # 示例：识别为"张三"
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── 李四/              # 示例：识别为"李四"
│   ├── face1.png
│   └── face2.png
└── 王五/               # 示例：识别为"王五"
    └── ...
```

## 图像要求

- **格式**：JPG、PNG 等常见图像格式
- **内容**：包含清晰人脸的正面照片
- **数量**：每人建议 3-10 张不同角度/光照的照片
- **命名**：文件名任意，文件夹名即为识别显示的姓名，仅支持英文

## 使用方式

上位机 OpenCV 程序会：

1. 遍历 `data/known/` 下的所有子文件夹
2. 读取每个文件夹内的人脸图像进行训练
3. 将文件夹名作为该人的识别标签
4. 检测到人脸后匹配并输出对应姓名
